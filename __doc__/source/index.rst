Tapyr documentation
===================

Welcome to tapyr's documentation!

This is the documentation for
`Christian Tanzer's <https://github.com/tanzer>`_
Python `repository <https://github.com/Tapyr/tapyr>`_
aka `Tapyr`. Tapyr is BSD licensed.

Tapyr comprises a number of :mod:`package namespaces
<_TFL.Package_Namespace>`:

- :mod:`MOM<_MOM>` implements a meta object model for the specification and
  implementation of essential object models.

- :mod:`GTW<_GTW>` implements a
  `Werkzeug-based <http://werkzeug.pocoo.org/>`_
  framework for RESTful web applications built on top of :mod:`MOM<_MOM>`.

- :mod:`JNJ<_JNJ>` provides a `Jinja-based <http://jinja.pocoo.org/>`_
  template framework for :mod:`GTW<_GTW>`.

- :mod:`CAL<_CAL>` provides classes for dealing with date, time, and
  calendars.

- :mod:`PMA<_PMA>` provides classes for dealing with emails.

- :mod:`ReST<_ReST>` provides
  `docutils-based <https://pypi.python.org/pypi/docutils/>`_
  classes and functions for dealing with
  re-structured text.

- :mod:`ATAX<_ATAX>` provides classes and functions for doing simple
  accounting according to Austrian tax laws.

- :mod:`TFL<_TFL>` provides utility classes and functions used by the other
  package namespaces.
